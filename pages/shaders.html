River Graphics Scene

About This Project

This scene is a simple simulation of a ravine, built using HLSL shaders in a pre-made DirectX 11 framework. The flowing water was simulated by applying gerstner waves to a plane which, while more applicable to an ocean than a river, provides an illusion of flowing water. The terrain was built by applying a heightmap to a plane and was textured using a texture map to blend two textures seamlessly.

The directional light is directed to the terrain from the yellow sphere that acts as a "sun," the position of which is arced by changing the "Time of Day" slider. The colour of the light also changes from a white to a white-yellow colour when the same slider is near either end to create a sunrise and sunset effect.

The scene also contains an edge detection post-processing effect, a depth map to show how the shadows from the directional and spotlights are calculated, and a drone model that can be fully manipulated to demonstrate the dynamic lighting.
