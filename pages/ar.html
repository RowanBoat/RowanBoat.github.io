Puck Launching AR Game

About This Project

Developed to be an Android mobile game that requires the user to move around the level displayed through AR to accurately move the puck object to the end marker.

This game was developed in Unreal Engine 5.0, using the engine's built-in AR functionaility. The puck launch is set up when the player taps and holds the screen and the launch then happens when the finger is released.

The force of the launch is determined based on how far the mobile device is moved from the initial tap position, and the direction of the force is determined by the orientation of the mobile device on the initial tap.
