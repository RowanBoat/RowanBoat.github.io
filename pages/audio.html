Cave Walking Simulator

About This Project

This project is a simple walking simulator set in a cave, built to demonstrate a dynamic soundscape. Built in the Unity Engine, the scene uses Steam Audio to create sounds in a 3D sound space based on its pre-built physics model. Some of the sounds in this scene were also recorded as foley, including the ambient effect, underwater ambient effect, the water footsteps, and the birds.
