Abertay Game Jam Manager 2023

About This Project

Developed as part of the 2023 Global Game Jam, I worked on this game as part of a team of three. The game is a very simple simulator where the player needs to keep the game jammers from becoming too rowdy, otherwise security won't allow another game jam to happen here. This is meant to be done by making sure their needs are met.

My contribution to the game was designing programming the UI, using assets provided by our artist. The UI was programmed in C# and designed using the Unity Engine UI tools and TextMeshPro assets.
